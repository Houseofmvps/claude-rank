# Support

## Where to get help

**GitHub Issues** is the right place for:
- Bug reports
- Feature requests
- Questions about behavior or output

Open an issue: https://github.com/Houseofmvps/claude-rank/issues

---

## Before opening an issue

1. Check the [README](./README.md) — most questions about commands and scoring are answered there.
2. Check [open and closed issues](https://github.com/Houseofmvps/claude-rank/issues?q=is%3Aissue) — your question may already be answered.
3. Make sure you are running Node.js >= 18 (`node --version`).

---

## Reporting a bug

Include:
- Node.js version
- claude-rank version (`npx claude-rank --version`)
- The command you ran
- Expected behavior
- Actual behavior or error output

---

## Security issues

Do not open a public issue for security vulnerabilities. See [SECURITY.md](./SECURITY.md) for the disclosure process.
